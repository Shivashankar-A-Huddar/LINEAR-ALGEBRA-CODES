Quick Reference Notes

Essential concepts explained clearly for ECE students. Tap any topic to expand.

S/V
Scalars vs Vectors
Magnitude only vs magnitude + direction

What is a Scalar?

A scalar quantity has magnitude only, with no direction. Most electrical quantities you measure with a multimeter are scalars.

  • Voltage (V), Current (I), Resistance (R), Power (P)
  • Temperature, Frequency, Capacitance, Energy
  • Represented as a single number: Q = 5 uC

What is a Vector?

A vector has both magnitude AND direction in space. Critical in EM field theory and antenna analysis.

  • Electric Field E, Magnetic Field B, Force F
  • Poynting Vector S, Current Density J
  • Represented as: v = [vx, vy] with magnitude |v| = sqrt(vx^2 + vy^2)
Scalar: Q = 1.5 uC (no direction needed) Vector: E = [3, 2] V/m, |E| = sqrt(9+4) = 3.6 V/m Force: F = Q x E (scalar x vector = vector)
V+
Vector Operations
Addition, subtraction, dot product, cross product

Vector Addition / Subtraction

Add or subtract component-wise. The result is always a vector.

A = [1, 2, 3], B = [4, 5, 6] A + B = [5, 7, 9] (Vector result) A - B = [-3, -3, -3] (Vector result) Scalar x A = 2 x [1,2,3] = [2, 4, 6]

Dot Product (Scalar result)

A . B = Ax*Bx + Ay*By. Result is a scalar. Used to find angle between vectors.

A . B = (1x4) + (2x5) = 4+10 = 14 (Scalar) cos(theta) = (A.B) / (|A||B|)

Cross Product (Vector result)

A x B gives a vector perpendicular to both. Essential in 3D EM fields.

KL
Kirchhoff's Laws
KVL: Sum(V)=0 | KCL: Sum(I_in) = Sum(I_out)

Kirchhoff's Voltage Law (KVL)

The algebraic sum of all voltages around any closed loop is zero. Energy is conserved.

Sum(V) = 0 (around any closed loop) Vs - V1 - V2 - V3 = 0 I = Vs / (R1 + R2 + R3) V1 = I*R1, V2 = I*R2, V3 = I*R3

Kirchhoff's Current Law (KCL)

The sum of currents entering a node equals the sum leaving. Charge is conserved.

Sum(I_in) = Sum(I_out) (at any node) I1 + I2 = I3 (example node) Node voltage: Vn = (G1*V1 + G2*V2 + G3*V3) / (G1+G2+G3)
DS
Discrete Signals
Sampling theory and array operations

Discrete vs Continuous Signals

A continuous signal has values at every instant of time. A discrete signal exists only at specific sample points n = 0, 1, 2, ...

  • Discrete signal: x[n] defined at integer indices
  • Obtained by sampling continuous x(t) at rate fs
  • Nyquist Theorem: fs >= 2 x fmax to avoid aliasing

Array Operations

x1[n] = [4, 7, 2], x2[n] = [2, 3, 7] Addition: x1 + x2 = [6, 10, 9] Subtraction: x1 - x2 = [2, 4, -5] Multiplication: x1 * x2 = [8, 21, 14]
